# 仿网易云音乐微信小程序

1. 感谢[NeteaseCloudMusicApi](https://binaryify.github.io/NeteaseCloudMusicApi)的接口。
2. 实现的功能：歌单、搜索、MV、视频、歌手排行、歌曲播放、视频播放、精品歌单等。
